Story mode UI (show when `isStoryMode === true`):
- Show a list of the user's completed sketches (from `useSketchContext().sketches` filtered by `status === 'completed' && animated_image_url !== null`) as selectable thumbnails — use `expo-image` for thumbnails
- A `TextInput` for story theme, placeholder "e.g. goes on a space adventure", max 200 chars
- A "Generate Story (5 credits)" button (disabled if no sketch selected or theme empty)

`handleGenerateStory()` function:
1. Validate: selectedSketchForStory and storyTheme both set
2. Check credits: show Alert if `profile.credits < 5`
3. Call `createStory(selectedSketchForStory, storyTheme)` — returns `{ storyId }` or null
4. On success: show `isProcessing=true` with "Generating your story..." text
5. Start polling: `storyPollIntervalRef.current = setInterval(pollStoryStatus, 10000)` (max 72 attempts = 12 min)
6. `pollStoryStatus()`: invoke `poll-story` with `{ storyId }` — check `data.status`:
   - `completed` → clear interval, setIsProcessing(false), toast "Your story is ready!", navigate to Gallery
   - `failed` → clear interval, setIsProcessing(false), toast "Story generation failed. No credits were charged."
   - `processing` → increment attempt counter; if > 72 → clear interval, setIsProcessing(false), toast "Generation is taking longer than expected. Check Gallery soon."
